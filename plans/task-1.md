# Task 1: Call an LLM from Code — Implementation Plan

## 1. Выбор LLM провайдера
- **Провайдер:** Qwen Code API
- **Модель:** coder-model (Qwen 3.5 Plus)
- **Причина:** 1000 бесплатных запросов в день, работает из России

## 2. Структура агента
- Чтение конфига из `.env.agent.secret`
- Использование OpenAI-совместимого клиента
- Запрос к LLM с системным промптом
- Вывод JSON с полями `answer` и `tool_calls`

## 3. Системный промпт
"Ты полезный ассистент. Отвечай кратко и по существу."

## 4. Тестирование
- Один регрессионный тест проверяет:
  - JSON-валидность ответа
  - Наличие полей `answer` и `tool_calls`
