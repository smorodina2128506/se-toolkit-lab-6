# Task 2: The Documentation Agent — Implementation Plan

## 1. Цель
Добавить инструменты list_files и read_file, чтобы агент мог читать файлы из wiki/ и отвечать на вопросы по документации.

## 2. Инструменты
- **list_files** — показывает список файлов в папке
- **read_file** — читает содержимое файла

## 3. Агентный цикл
- Отправляю запрос в LLM с описанием инструментов
- Если LLM хочет вызвать инструмент — вызываю, добавляю результат в историю
- Повторяю, пока не получу ответ без tool_calls (максимум 10 раз)

## 4. Безопасность
Проверяю, что путь начинается с корня проекта (защита от ../)

## 5. Модель
Использую ту же: coder-model через Qwen API
